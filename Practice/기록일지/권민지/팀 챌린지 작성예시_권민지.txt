팀 챌린지 : 1일 1 commit 하기
내용 상관없이 매일 commit을 해보는 팀 챌린지를 해보고자 합니다
작성 방법은 
github desktop을 이용하시거나 아주 간단하게는 github 사이트에서
자신에게 맞는 폴더로 들어가 수업 시간에 자습 후 제출하는 텍스트 파일을 올리시거나
직접 간단하게 작성해주시면 됩니다

내용은 확인 할 것도 아니고 크게 중요하지 않으며

이 챌린지의 의의는 commit에 익숙해지는 것과
현업을 위해 commit시 제목과 내용을 알아보기 쉽게 작성해보는 연습을 해보는 것이며
github 잔디밭을 파이널 기간 동안 만들어보고자 하는 도전이기도 합니다.

부담 없는 선에서 챌린지를 도전하고자 하니 팀원분들의 많은 참여 바랍니다.
처음엔 익숙하지 않고 왜하나 싶을지 모르지만 저희 파이널 팀프로젝트에도 도움이 될 것이 분명하고
개인적으로도 github를 사용하시는데 익숙해지실 수 있다고 생각합니다

챌린지 초반에는 그냥 master에 올리는 걸 목표로 하고 있습니다.
최대한 쉽고 간편하게 commit을 꾸준히 하는것에 집중해보도록 해봅시다

수업이 끝나는 21 ~ 21시 50분 사이에 선생님께 공부한 내용을 제출하시게 될텐데
해당 파일 또는 그거 하시는김에 간단히 작성을 해주시면 될 것 같습니다
